1. Overview

KydeOS is a free, open-source operating system designed and maintained by a single independent developer. This Privacy Policy explains what data KydeOS collects during normal system operation, how that data is handled, and what rights you hold as a user of this software.

The guiding principle behind KydeOS's approach to privacy is straightforward: collect the absolute minimum necessary, be fully transparent about what is collected, and never share user data with third parties under any circumstances. KydeOS is not a commercial product. There are no investors, no advertisers, no monetization strategies dependent on user data. This project exists solely as a personal, open-source endeavor.

Because KydeOS is fully open source under the GPLv3, every line of code that runs on your machine is publicly available and auditable by anyone. This means you are never required to take our word for it — you can verify exactly what KydeOS does and does not do with your data by reading the source code directly.

2. What Data KydeOS Collects

Under normal operating conditions, KydeOS collects no personal data whatsoever. No telemetry pipelines exist in the base system. No crash reports are transmitted automatically. No usage analytics, behavioral metrics, hardware fingerprints, or diagnostic data of any kind are sent to any remote server without your explicit, informed consent.

The following specific scenarios may involve limited and unavoidable data exposure:

  • System update checks: When KydeOS checks for available system updates, a standard network request is made to the update server. This request will expose your IP address to that server as a natural consequence of how internet networking works. No other data is transmitted alongside this request. Your IP address is not logged, stored, or cross-referenced with any user account or identity.
  • KydeStore — developer and alpha tester registration: KydeStore is the native software center built into KydeOS. An alpha developer program is available, allowing early community members to register as testers. If you choose to register for this program inside KydeStore, the information you voluntarily submit during registration — which may include a username, email address, or similar identifiers — is collected solely for the purpose of managing that program. This information is not sold, shared with third parties, used for advertising, or processed for any purpose beyond the developer program itself.
  • Third-party applications: KydeOS supports installation of software via .deb packages, Flatpak, and AppImage. Applications installed through these formats are entirely independent software projects with their own codebases, developers, and privacy practices. KydeOS has no visibility into, control over, or responsibility for the data practices of any third-party application you choose to install. You should consult the privacy policy of each application individually.
  • Krowser — the built-in browser: Krowser is a first-party browser developed by the KydeOS developer and bundled as the default browsing experience. When you use Krowser to browse the web, it connects to the websites you visit. Those websites operate under their own privacy policies and may collect data according to their own practices. Krowser does not transmit your browsing history, search queries, visited URLs, or any usage data back to the KydeOS developer. Krowser is a browser, not a surveillance tool.

3. What Data KydeOS Does Not Collect

To be fully explicit, KydeOS does not collect, transmit, store, or process any of the following:

  • Full names, email addresses, or any personally identifiable information during normal OS operation
  • Hardware identifiers, device serial numbers, or machine fingerprints
  • Geographic location data of any kind
  • Usage statistics, session durations, or behavioral analytics
  • Crash reports or diagnostic logs transmitted automatically
  • Keyboard input, clipboard contents, or on-screen activity
  • Installed application lists
  • Network activity logs or browsing history
  • Any data sold to, shared with, or accessible by advertisers, data brokers, or any commercial third party

4. Open Source as a Privacy Guarantee

KydeOS is licensed under the GNU General Public License v3.0. This is not merely a legal formality — it is a structural privacy guarantee. Every component of KydeOS that runs on your machine is publicly available for inspection at https://github.com/r1nzd/KydeOS.

Anyone with the technical knowledge to do so can audit the codebase to verify that no hidden telemetry, background data collection, or undisclosed network communication exists anywhere in the system. This level of transparency is not possible with closed-source operating systems, and it is one of the core reasons KydeOS is and will remain fully open source.

If you discover any behavior in KydeOS that appears to contradict this policy — any unexpected network request, any undisclosed data transmission, or any privacy-relevant code that is not documented here — please report it immediately as a security or privacy issue on the GitHub repository. Such reports are taken with the utmost seriousness.

5. Security

Although KydeOS does not collect personal data, the project takes system-level security seriously. The following security features are included in KydeOS:

  • LUKS2 full-disk encryption support, protecting all data stored on your device at rest
  • AppArmor and SELinux sandboxing to isolate applications and limit potential damage from compromised software
  • A minimal default service configuration, reducing the attack surface by avoiding unnecessary background daemons and exposed ports
  • Regular security patches applied from the Debian Stable upstream, with critical fixes applied out-of-cycle whenever necessary

6. Your Rights

Because KydeOS does not collect personal data during normal system operation, there is in most cases no personal data held by the developer that pertains to you. If you participated in the KydeStore developer program or interacted with any associated service that did collect information from you, you have the right to:

  • Request access to any personal data held about you
  • Request correction of inaccurate data
  • Request deletion of your data at any time

To exercise any of these rights, contact the developer via GitHub at https://github.com/r1nzd/KydeOS.

7. Children's Privacy

KydeOS is not directed at children under the age of 13 and does not knowingly collect any personal data from minors. If you have reason to believe that a minor has submitted personal information through KydeStore or any associated service, please contact the developer via GitHub so that the information can be promptly removed.

8. Changes to This Policy

This Privacy Policy may be updated as KydeOS evolves — as new features are introduced, new services are launched, or new legal requirements arise. All changes will be committed to the public GitHub repository with a clear and auditable diff, so the full history of this document is always publicly accessible.

The effective date at the top of this document will be updated to reflect the date of the most recent revision. Continued use of KydeOS following the publication of changes constitutes acceptance of the updated policy.
